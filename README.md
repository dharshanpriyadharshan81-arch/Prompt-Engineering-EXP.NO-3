EX.NO:3 Exploring the different types of prompts- Straightforward Prompts - Tabular Format Prompting - Missing Word Prompting - Preceding Question Prompting.

Aim: To write the prompts for the following prompt types for 2 different scenarios of Each type.

1.Straightforward Prompts

2.Tabular Format Prompting

3.Preceding Question Prompting

4.Missing Word Prompting

Explanation - Any one use case from Unit 5 or any 2 own use cases with the unit 2 Prompt type

Procedure:

Write 2 different prompts for different use cases, refer the following Examples

Example:

Straightforward Prompts:

• "Define photosynthesis in one sentence."

"List three advantages of electric vehicles."

Tabular Format Prompting:

• "Compare and contrast AC and DC current in a table."

• "Provide a table listing five programming languages, their paradigms, and one use case each."

Preceding Question Prompting:

• "Why is climate change a global concern? Explain how greenhouse gases contribute to global warming."

• "How do vaccines work? Describe the process of immunization in simple terms."

Missing Word Prompting:

• "The capital of France is ____."

• "In photosynthesis, plants absorb sunlight to produce ____."

OUTPUT:

1. Straightforward Prompts

A direct instruction asking the AI to do something.

Example 1

Prompt:
Explain the causes and effects of climate change in simple terms.

Example 2

Prompt:
Write a short paragraph explaining the importance of renewable energy.

2. Tabular Format Prompting

The prompt asks the AI to present information in a table.

Example 1

Prompt:
Create a table comparing Solar Energy, Wind Energy, and Hydro Energy with columns for Source, Advantages, and Disadvantages.

Example 2

Prompt:
Provide a table showing three programming languages (Python, Java, C++) with columns for Developer, Year Created, and Main Use.

3. Preceding Question Prompting

A question is asked before the main task to guide the response.

Example 1

Prompt:
What are the main functions of the human heart?
After explaining, describe how the heart helps in blood circulation.

Example 2

Prompt:
What is Artificial Intelligence?
Based on your explanation, list three real-life applications of AI.

4. Missing Word Prompting

A sentence is given with missing words that the AI must complete.

Example 1

Prompt:
Fill in the blanks:
The process by which plants make their food using sunlight is called ______.

Example 2

Prompt:
Complete the sentence:
The capital city of India is ______ and it is located in the ______ part of the country.
